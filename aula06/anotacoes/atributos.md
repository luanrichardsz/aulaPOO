# Atributos

São propriedades ou elementos que representam o estado de um determinado objeto <br>
- Ex.: <br>
Cliente<br>
    - Nome;<br>
    - CPF;<br>
    - RG;<br>
    - Endereço;<br>
    - Estado Civil;<br>
    - ...<br>

- Declaração de Atributos:<br>
- private static final int TAMANHO = 200;<br>
- private int indice = 0;<br>
- private String nome, cpf;<br>
